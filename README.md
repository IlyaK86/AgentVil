# Vilavi Task Intelligence Dashboard

Interactive dashboard for Vilavi Bitrix24 task management.

## Features

- Task statistics overview
- Problematic task detection
- User performance analysis
- Dead task tracking (>10 days without movement)
- Overdue task monitoring

## Usage

Open `index.html` in a browser.

## Data Source

Dashboard connects to Vilavi Task Intelligence API via Python scripts.
